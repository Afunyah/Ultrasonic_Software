Private Member Functions

void CalculateTrajectoryTransducerData ()
 From the nearby transducers on a given path, calculate near and far transducers to be used by the chosen phase and amplitude manipulation method. This takes into account two ghost particles in sequence, allowing for the future moves of the particle to be predicted The transducers involved for each ghost particle are separated, with the intersecting transducers being allocated to the nearer ghost Adds a tuple of lists to the TrajectoryTransducerData The first element contains GhostTransducerPositionData pertaining to 'Postion 2, Transducer 1' The second element contains GhostTransducerPositionData pertaining to 'Postion 2, Transducer 2'.
